Invariants

  • Materialize creates an inactive Draft only — never writes runtime context.
  • Activate is the single bridge that affects runtime behavior, and requires an explicit human click.
  • Curator and Queue cannot reach the Active Surface directly. Dashboard review is mandatory.
  • One canonical schema: CandidateV1. The Schema Gate rejects malformed proposals.
  • Legacy 3.0 / incompatible payloads are batch-quarantined; no compatibility import.
  • No auto-load of raw observations; no auto-promote of candidates.
